# 📁 Models - Estructura Física del Vehículo

En esta sección se documenta el diseño y montaje del chasis de nuestro robot autónomo para la eliminatoria regional.

* **Chasis:** Estructura vehicular comercial de 4 ruedas con sistema de dirección delantera Ackermann y tracción trasera mediante un motor DC.
* **Soporte de Sensores:** Base frontal personalizada fabricada artesanalmente en madera, diseñada a medida para alojar los dos sensores ultrasónicos fijos orientados en diagonal hacia los laterales.
* **Dimensiones:** El chasis y sus componentes añadidos cumplen estrictamente con las medidas máximas reglamentarias exigidas por la WRO (30x30x30 cm).
